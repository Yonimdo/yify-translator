Next Steps

  • If you are following the Getting Started guide, then please head back to the tutorial and follow the next steps!
  • If you wat to generate a new key from the Lengua CLI, run:
    • git clone https://github.com/syrmapp/lenuga-getting-started.git - this will create a local copy of the source code for the app
    • cd lenuga-getting-started - change directory into the local source code repository
    • lengua git:remote -a <your-app-name> - associate the Lenuga app with the repository
    • You'll now be set up to run the app locally, or configure changes to Lengua app.
